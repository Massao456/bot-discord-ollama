# Bot de conversa (Discord)

Projeto simples em Node.js de um bot de IA

**Stack**
- `Node.js` (v16)
- `discord.js` (v14.25.1)
- `dotenv` (v17.2.3)
- `ollama` (v0.6.3)


**Requisitos**
- Node.js v16+ (recomendado)
- npm ou yarn

**Instalação**
1. Instale dependências:

```powershell
npm install
```

2. Crie um arquivo `.env` na raiz com as variáveis necessárias (ex.: `DISCORD_TOKEN`, `OLLAMA_API_KEY`).

**Execução**
Execute o bot com:

```powershell
node app.js
```
Ou executo com:

```powershell
npm start
```
